# Willkommen zur Dokumentation

Willkommen auf der CanSat NeXT-Dokumentationsseite! Diese Seite enthält detaillierte technische Dokumentationen der CanSat NeXT-Hardware und -Software sowie leicht verständliche Tutorials zum Einrichten Ihres CanSat NeXT und zur Nutzung der verschiedenen Hardware-Funktionen auf der Platine.

Wenn Sie neu hier sind, gehen Sie zur [Erste Schritte-Seite](./course/course.md) für Informationen, wie Sie mit CanSat NeXT beginnen können. Vielleicht interessieren Sie sich auch für Informationen zur Herstellung der Basisantenne aus den im Kit enthaltenen Materialien. Dafür gehen Sie zum Artikel [Kommunikation und Antennen](./CanSat-hardware/communication).

Während Sie hier sind, vergessen Sie nicht, auch unseren [Blog](./../blog/) zu besuchen, in dem wir Projekte zeigen, die CanSat NeXT für Zwecke jenseits von CanSat nutzen. Diese sind keine CanSat-Projekte, zeigen aber dennoch die Möglichkeiten, was mit CanSat NeXT gemacht werden kann.

Schließlich, wenn Sie so sind wie ich, sind Sie wahrscheinlich auf diese Seite gekommen, um nach [der Bibliotheksspezifikation](./CanSat-software/library_specification) oder dem Pinout zu suchen. Mehr Informationen dazu finden Sie unter [Pinout](./CanSat-hardware/pin_out), aber hier ist eine kurze Referenz:

# Lizenz

Diese Bibliothek und die CanSat NeXT-Platine werden von Samuli Nyman in Zusammenarbeit mit ESERO Finnland und Arctic Astronautics Oy entwickelt. Die Entwicklung wird auch von der Finnischen Physikalischen Gesellschaft unterstützt. Diese Softwarebibliothek ist unter der MIT-Lizenz lizenziert.
