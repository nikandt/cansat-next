# Bienvenido a la Documentación

¡Bienvenido a la página de documentación de CanSat NeXT! Este sitio incluye documentación técnica detallada del hardware y software de CanSat NeXT, así como tutoriales fáciles de abordar sobre cómo configurar tu CanSat NeXT y utilizar las diversas características de hardware en la placa.

Si eres nuevo aquí, dirígete a la [Página de Introducción](./course/course.md) para obtener información sobre cómo comenzar a usar CanSat NeXT. También puede interesarte la información sobre cómo hacer la antena básica con los materiales incluidos en el kit. Para eso, dirígete al artículo [Comunicación y Antenas](./CanSat-hardware/communication).

Mientras estás aquí, no olvides también revisar nuestro [blog](./../blog/), donde mostramos proyectos que utilizan CanSat NeXT para propósitos más allá de CanSat. Estos no son proyectos de CanSat, pero aún así muestran las posibilidades de lo que se puede hacer con CanSat NeXT.

Finalmente, si eres como yo, probablemente viniste a esta página buscando [la especificación de la biblioteca](./CanSat-software/library_specification) o el pinout. Más información sobre eso en [Pinout](./CanSat-hardware/pin_out), pero aquí tienes una referencia rápida:
