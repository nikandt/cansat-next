# Bem-vindo à Documentação

Bem-vindo à página de documentação do CanSat NeXT! Este site inclui documentação técnica detalhada do hardware e software do CanSat NeXT, bem como tutoriais fáceis de seguir sobre como configurar seu CanSat NeXT e usar os vários recursos de hardware da placa.

Se você é novo aqui, vá para a [Página de Introdução](./course/course.md) para obter informações sobre como começar a usar o CanSat NeXT. Você também pode se interessar por informações sobre como fazer a antena básica com os materiais incluídos no kit. Para isso, vá para o artigo [Comunicação e Antenas](./CanSat-hardware/communication).

Enquanto estiver aqui, não se esqueça de conferir nosso [blog](./../blog/), onde mostramos projetos usando o CanSat NeXT para fins além do CanSat. Estes não são projetos de CanSat, mas ainda assim mostram as possibilidades do que pode ser feito com o CanSat NeXT.

Finalmente, se você é como eu, provavelmente veio a esta página procurando pela [especificação da biblioteca](./CanSat-software/library_specification) ou pelo pinout. Mais informações sobre isso em [Pinout](./CanSat-hardware/pin_out), mas aqui está uma referência rápida:
